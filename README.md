# 实验室出入库系统微信小程序

一个可直接导入微信开发者工具的小程序示例，包含：

- 物资出入库管理
- 大型设备使用记录
- 仪器使用记录
- 试剂使用记录
- 危险品标识与分类（有毒有害、易燃易爆、其他）
- AI 助手连续对话页（多轮补充草稿、候选物品确认、确认卡片统一入账）

## 目录结构

- `miniprogram/` 小程序前端
- `cloudfunctions/` 预留云函数目录
- `cloudfunctions/aiAssistant/` AI 助手云函数与提示词配置

## 当前实现

- 使用 `wx.setStorageSync` 做本地持久化，打开即可演示
- 首页展示库存、危险品、低库存和近期动态
- 出入库页支持入库/出库登记并自动更新库存
- 设备使用页支持新增大型设备使用记录
- 仪器试剂页支持新增仪器使用记录和试剂使用记录
- AI 助手页支持连续对话整理草稿，确认后批量写入现有库存台账

## 使用方式

1. 用微信开发者工具打开目录 `C:\Users\29035\Desktop\小程序测试`
2. 如果你有自己的小程序 AppID，把 `project.config.json` 里的 `appid` 改成真实值
3. 在云开发环境中创建并部署 `cloudfunctions/aiAssistant/` 云函数
4. 为云函数配置环境变量：
   - `AI_API_KEY`：第三方大模型 API Key
   - `AI_BASE_URL`：OpenAI 兼容接口基础地址，例如 `https://xxx/v1`
   - `AI_MODEL`：模型名
5. 如需修改提示词，可直接编辑 `cloudfunctions/aiAssistant/config.js`
6. 重新编译后，从 tabBar 进入 “AI助手” 页面预览
